<h2>Teng sonli belgilar to’plami</h2>
<p>Agar satrda barcha simvollar bir xil sonda qatnashgan bo’lsa bu satrteng sonli belgilar to’plamideyiladi.

<h2>Kiruvchi ma'lumotlar:</h2>

<h2>Chiquvchi ma'lumotlar:</h2>
