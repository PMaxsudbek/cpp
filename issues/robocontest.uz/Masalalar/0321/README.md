<h2>To’plam unvoni</h2>
<p>Sizga N ta elementdan iborat A to’plam berilgan. Siz ixtiyoriy marotaba A to’plamning ixtiyoriy elementini qiymatini 1 ga kamaytirishingiz mumkin.
