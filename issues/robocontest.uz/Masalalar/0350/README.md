<h2>Shaxmat musobaqasi 2</h2>
<p>Adiz, Laziz va Bobur shaxmat musobaqasini o’tkazishdi. Bobur hakamlik qildi, Adiz va Laziz esa jami N ta partiya o’ynashdi. <h2>Chiquvchi ma'lumotlar:</h2>
