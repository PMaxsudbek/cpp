<h2>Kompleks sonlar (ko'paytma)</h2>
<p>Kompleks sonlar to'plamihaqida eshitgan bo'lsangiz kerak. Siz ushbu ifodani hisoblashingiz kerak.(Tushunmagan bo'lsangiz misollarga qarang!).</p>
