<h2>Number + Frequency</h2>
<p>Uzunligi\(m\)bo`lgan butun sonli\(a\)massivning bahosi deb quyidagi summaga aytiladi:

Sizga uzunligi\(n\)bo`lgan butun sonli\(b\)massiv hamda\(q\)ta so`rov beriladi.
