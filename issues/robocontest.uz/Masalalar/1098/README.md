<h2>1100 km</h2>

Radar yonidan o'tayotganda tezlik K\(km/soat\)dan oshmasligi lozim. Mashina maksimal tezlanishi(tormozlanishi)Â A\(km/soat^2\)va maksimal tezlik V\(km/soat\).
Shaharga qanday tezlikda kirish ahamiyatsiz va boshlang'ich tezlik nolga teng.</p>
