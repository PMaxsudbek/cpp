<h2>Maktabimizdagi o'yin</h2>
<p>Mirzo Ulug'bek nomidagi ixtisoslashtirilgan maktabida o'quvchilar zerikib qolmasliklari uchun, bir qancha o'yinlar uyushtirilib turiladi. Ulardan biri:
