<h2>Epik son</h2>
<p>Shohruh epik sonlarni juda ham yaxshi ko'radi. Endi u N sonini epik usulda yaratmoqchi. N sonini epik usulda yaratishlar sonini hisoblang.

<h2>Kiruvchi ma'lumotlar:</h2>

<h2>Chiquvchi ma'lumotlar:</h2>
